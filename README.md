
# @tradle/bot-style-me

## Usage

Set the correct provider URL in [conf.json](./conf.json)

```sh
yarn start
```

Along with your other strategies:

```js
const styleMe = require('@tradle/bot-style-me')
bot.use(styleMe)
```
